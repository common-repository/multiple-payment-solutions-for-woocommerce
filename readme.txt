== Description ==

Multiple Payment Solutions for Woocommerce is WordPress plugin , which gives you option to take payment through your woocommerce store , Currently we have built in options for Paytm (Merchant) , InstaMojo , PayUMoney, Cashfree.




== Features ==
*	Enable Payment support for InstaMojo - Paytm & PayuMoney
*	Support UPI / QR Code Payment as well.
* Custom 'THANK YOU' page.
* Customized “Redirect to payment page” message.
*	Support Order Refund Option [ Live Mode ]
* Mode of transaction Sandbox and Live

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/multiple-payment-solutions-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin

NOTE : This plugin is based on the Official API's which is provided by the supported Payment Gateways. The terms & services for each payment gateway are listed below.

==  Terms & Services for the supported Payment Providers ==
1 ) [Instamojo](https://www.instamojo.com/terms/)
2)  [Paytm](https://pages.paytm.com/terms.html)
3)  [PayUMoney](https://www.payu.in/tnc)
4)  [Cashfree]https://www.cashfree.com/tnc
